# Little Lemon Booking App

A React application for booking restaurant reservations where the user can select a date, a time, number of guests and an occasion for the reservation.

## Features
- Controlled booking form
- Dynamic time selection using reducer
- Form validation
- Confirmation page
- Unit tests

## Technologies
- React
- React Router
- JavaScript
- CSS
- Jest & React Testing Library

## Getting Started

git clone https://github.com/jennh8/little-lemon.git
cd little-lemon
npm install
npm start

## Running Tests

npm test
